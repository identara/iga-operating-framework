# Companion artifacts

Companions are non-normative. They operationalize the core's decisions, and where a companion and a chapter diverge, the chapter governs. Each companion is a file in this directory.

- Program charter template: `iga-program-charter-template.md` (chapter 1)
- Decision-rights and responsibility matrix starter: `iga-raci-starter-template.md` (chapter 2)
- Scope register template: `iga-scope-register-template.md` (chapter 3)
- Tier criteria and on-ramp worksheet: `iga-tier-and-onramp-worksheet.md` (chapter 4)
- Lifecycle definition checklist: `iga-lifecycle-checklist.md` (chapter 5)
- Certification campaign checklist: `iga-certification-checklist.md` (chapter 5)
- Cadence table template: `iga-cadence-table-template.md` (chapter 6)
- Platform capability checklist: `iga-platform-capability-checklist.md` (cross-cutting)
- Interactive framework explorer, a single self-contained page suitable for GitHub Pages: `framework-explorer.html`

Each worksheet also ships as a fillable Word document in `docx/`.
