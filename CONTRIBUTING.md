# Contributing

Archetype profiles are the primary contribution surface; the bar a profile must meet is AP1 to AP5 in `profiles/`. Comments on any part of the framework go through repository issues, citing statement identifiers (M1, O1, S1, P1, PR1, C1, MS1, AP1) so discussion stays precise. Language conventions and reading paths are in the repository README.
